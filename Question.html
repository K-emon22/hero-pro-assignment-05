Question-1: What are the different ways to select an element in the DOM?
The different ways to select an element in the DOM are, getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector() and querySelectorAll()
Question-2: What is the difference between innerHTML, innerText, and textContent ?
The difference between innerHTML, innerText, and textContent are, innerHTML change or catch certain containers inside html or tag, innerText change or catch certain containers innner text and textContent change or catch all text including hidden text.
Question-3: What is event delegation in the DOM?
event delegation in the DOM is a thing in javaScrip that parent element is used to handle events on its child elements.
Question-4: What is event bubbling in the DOM?
Event bubbling is a process when event happen on a element, it propagates up to all parent elements until it reach document.
Question-5: How do you create, add, and remove elements using JavaScript?
I will create an element using document.createElement(tag name), add element using parent.appendChild(child), remove element using parent.removeChild(child).
